viddec.h File Reference

Detailed Description

The VIDDEC video decoder interface. Provides the user an interface to create and interact with XDAIS algorithms that are compliant with the XDM-defined IVIDDEC video decoder interface.

#include <ti/xdais/dm/xdm.h>
#include <ti/xdais/dm/ividdec.h>
#include <ti/sdo/ce/Engine.h>
#include <ti/sdo/ce/visa.h>
#include <ti/sdo/ce/skel.h>
