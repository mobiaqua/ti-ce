sphenc1.h File Reference

Detailed Description

The SPHENC1 speech encoder codec interface. Provides the user an interface to create and interact with XDAIS algorithms that are compliant with the XDM-defined ISPHENC1 speech encoder interface.

#include <ti/xdais/dm/xdm.h>
#include <ti/xdais/dm/isphenc1.h>
#include <ti/sdo/ce/Engine.h>
#include <ti/sdo/ce/visa.h>
#include <ti/sdo/ce/skel.h>
